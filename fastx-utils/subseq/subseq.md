# fastx-utils之获取序列集合的子集或补集：subseq

### 一、fastx-utils subseq介绍

**功能描述：**

`fastx-utils subseq`根据提供的序列 `ID`，获取序列集合的交集序列或者补集序列。

**命令行接口：**

    $ fastx-utils subseq
    
    Usage: fastx-utils subseq [options] <in.fa/fq> <name.list>
    
    Options:
      -s   supplementary set option.

**可选参数：**

    -s      补充设置选项；
